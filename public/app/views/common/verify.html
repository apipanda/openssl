Verify

We verify that you own your domain — that’s your business web address, like yourcompany.com — to ensure that no one else claims ownership or generates certificates without your permission.

To verify your domain, select a verification option from below.

Verify With TXT Record

To verify ownership of your domain, you'll need to sign in to your domain host, which is most likely the service you used to buy your domain name.
  1. Sign in to your domain's host account.
  2. Locate the page for updating your domain's DNS records.
    The page might be called something like DNS Zone File, Name Server Management, or Advanced Settings.
  3. Create a new TXT record and set it to this value:
    openssl-domain-verification=.
  4. Click the Verify button below.

Verify With CNAME Record

To verify ownership of your domain, you'll need to sign in to your domain host, which is most likely the service you used to buy your domain name. You add CNAME records using the administration tools at your domain host. The exact steps for your host might vary. Below you can find generic steps that apply for any host as well as specific instructions for several popular hosts.

Adding a CNAME record does not disrupt your current mailflow or affect other services.
CNAME record changes can take up to 72 hours to go into effect, but typically they happen much sooner.

  1. Sign in to your domain's account at your domain host.
  2. Locate the page for updating your domain's DNS records.
    The page might be called something like DNS Zone File, Name Server Management, or Advanced Settings.
  3. Locate the CNAME records for your domain.
  4. Add a CNAME record with the Name/Alias/Host value set to . and the Value/Destination/Answer field set to openssl.io.
  5. Save your changes and wait until they take effect.
  6. Click the Verify button below.

Upload file to your webserver

You can verify your domain ownership by uploading an HTML file to your domain’s website. When you upload the HTML file to your web server, we check that it’s there. Uploading the file doesn’t affect your website.

The HTML file must be available at your naked domain, which means it doesn’t have www in the URL. For example, http://yourdomain.com/yourfile.html is a naked domain.
The HTML file must be publicly available and not limited to an internal network (an intranet, for example).
Note: Do not rename your HTML file or we won’t recognize it when we look for it.

  1. Download your HTML verification file
  2. Upload the HTML file to the root folder (top-level folder) of your domain’s web server. The root folder's name can vary by hosting provider—it's typically public_html, www, or wwwroot.
    Note: Do not upload the HTML file to a subfolder. The file must be visible at http:///.html.
  3. Open a new browser window and navigate to http:///.html
    Note: If the file uploaded correctly, you'll see in your browser openssl-domain-verification=.
  4. Click the Verify button below.

Add META TAG to your homepage

We will give you a meta tag containing a verification code to paste into the section of your website’s homepage. Visitors to your website won't see the meta tag, but we can check that it's there.

The HTML file must be available at your naked domain, which means it doesn’t have www in the URL. For example, http://yourdomain.com/ is a naked domain.
The website must be publicly available and not limited to an internal network (an intranet, for example).
Note: Do not rename your HTML file or we won’t recognize it when we look for it.

  1. Open your website homepage HTML file using any text or HTML editor. If your web host uses a web-based page editor (Squarespace, Wix, WordPress.com, and so on), open your homepage in their editor.
  2. Paste this meta tag <meta name="openssl-domain-verification" content="openssl-domain-verification=" /> into the <head> section that’s before the first <body> section. If you’re unsure how to add meta tags with a web-based page editor, contact your web host’s technical support.
  3. Save the changes to your home page. If you edited the file offline, upload it to your web server, overwriting the previous homepage file.
  4. Click the Verify button below.
